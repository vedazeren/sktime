sktime.utils.time_series.add_trend(x, coefs, time_index=None)[source][source]¶

Add trend to array for given fitted coefficients along axis 0 or 1, inverse function to remove_trend()

Parameters

  • x (array_like, shape=[n_samples, n_obs]) – Time series data, each sample is treated separately

  • coefs (array-like, shape=[n_samples, order + 1]) – fitted coefficients of polynomial order for each sample, one column means order zero, two columns mean order 1 (linear), three columns mean order 2 (quadratic), etc

  • time_index (array-like, shape=[n_obs], optional (default=None)) – Time series index for which to add the trend components

Returns

xt – The series with added trend.

Return type

ndarray

See also

fit_trend(), remove_trend()

sktime.utils.time_series.fit_trend(x, order=0)[source][source]¶

Fit linear regression with polynomial terms of given order

x : array_like, shape=[n_samples, n_obs] Time series data, each sample is fitted separately

orderint

The polynomial order of the trend, zero is constant (mean), one is linear trend, two is quadratic trend, and so on.

Returns

coefs – Fitted coefficients of polynomial order for each sample, one column means order zero, two columns mean order 1 (linear), three columns mean order 2 (quadratic), etc

Return type

ndarray, shape=[n_samples, order + 1]

See also

add_trend(), remove_trend()

sktime.utils.time_series.remove_trend(x, coefs, time_index=None)[source][source]¶

Remove trend from an array with a trend of given order along axis 0 or 1

Parameters

  • x (array_like, shape=[n_samples, n_obs]) – Time series data, each sample is de-trended separately

  • coefs (ndarray, shape=[n_samples, order + 1]) – Fitted coefficients for each sample, single column means order zero, two columns mean order 1 (linear), three columns mean order 2 (quadratic), etc

  • time_index (array-like, shape=[n_obs], optional (default=None)) – Time series index for which to add the trend components

Returns

xt – The de-trended series is the residual of the linear regression of the data on the trend of given order.

Return type

ndarray

See also

fit_trend(), add_trend()

References

Adapted from statsmodels (0.9.0), see https://www.statsmodels.org/dev/_modules/statsmodels/tsa/tsatools.html #detrend
